# Filosofia do PCBoostLab

## Objetivo

Criar um software profissional para diagnóstico e otimização do Windows com foco em desempenho, estabilidade e segurança.

---

## Princípios

- Interface simples.
- Interface rápida.
- Nunca travar.
- Sempre usar carregamento assíncrono quando necessário.
- Explicar tudo ao usuário.
- Priorizar segurança.
- Não realizar alterações perigosas automaticamente.
- Sempre informar riscos.
- Sempre permitir desfazer alterações quando possível.

---

## Experiência do usuário

- Nada deve parecer travado.
- Sempre mostrar loading.
- Sempre mostrar progresso.
- Sempre mostrar mensagens amigáveis.

---

## Desenvolvimento

- Código modular.
- Código organizado.
- Funções pequenas.
- Comentários apenas quando necessários.
- Git organizado.
- Commits pequenos.

---

## Performance

- Evitar consumo excessivo de RAM.
- Evitar consumo desnecessário de CPU.
- Não executar leituras repetidas sem necessidade.

---

## Segurança

- Nunca modificar BIOS.
- Nunca alterar Registro automaticamente sem confirmação.
- Nunca remover arquivos críticos.
- Nunca desativar recursos de segurança do Windows sem explicar consequências.

---

## Git

Toda alteração relevante deve ser:

- commitada
- documentada
- enviada ao GitHub

antes da próxima funcionalidade.
