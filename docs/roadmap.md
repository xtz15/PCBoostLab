# PCBoostLab - Roadmap

## Objetivo

Este documento define a ordem oficial de desenvolvimento do PCBoostLab.

Novas funcionalidades só devem ser adicionadas ao roadmap após aprovação.

---

# Versão atual

## v0.1 ✅

Concluído

- Estrutura modular
- Interface principal
- Painel
- Diagnóstico de CPU
- Diagnóstico de RAM
- Diagnóstico do Sistema
- Diagnóstico de Processos
- Diagnóstico de Discos
- Relatórios TXT
- Interface assíncrona
- Loading nas telas
- Integração com GitHub
- Documentação inicial

# Próximas versões

## v0.2 — Limpeza Segura

Objetivos:

- Analisar arquivos temporários
- Limpar arquivos temporários do usuário
- Limpar arquivos temporários do Windows
- Limpar cache de miniaturas
- Limpar Prefetch (com aviso)
- Abrir Limpeza de Disco do Windows
- Abrir "Limpar arquivos do sistema"
- Mostrar estimativa de espaço recuperado
- Mostrar resumo da limpeza
- Registrar resultado em relatório
